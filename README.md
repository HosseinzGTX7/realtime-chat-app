# Realtime Chat App 💬

A simple real-time chat app built with **Node.js** and **Socket.IO**.  
This project was created to practice building realtime applications and working with WebSockets using Socket.IO.

## Tech Stack 🛠️

- Node.js
- Express.js
- Socket.IO
- HTML
- CSS
- JavaScript

## Features ✨

- Real-time messaging
- Live updates without page refresh
- Username customization
- Typing status indicator
- Browser notifications for new messages
- Message send sound effects
- Connected clients counter
- Socket connection logs in terminal
- Chat UI inspired by popular messaging apps
- Message timestamps
- Lightweight and simple architecture

## Installation 📥

Clone the repository:

```bash
git clone https://github.com/HosseinzGTX7/realtime-chat-app.git
cd realtime-chat-app
```

Install dependencies:

```bash
npm install
```

You can also use:

```bash
yarn install
```

## Running the Project ▶️

Start the server:

```bash
node app.js
```

Or:

```bash
npm start
```

After running the server, a local port will be displayed in the terminal.  
Open it in your browser to start chatting.

Example:

```bash
http://localhost:4000
```

## Main Dependencies 📚

- express
- socket.io

## Realtime Events ⚡

The application updates instantly without refreshing the page for events such as:

- Sending messages
- Username changes
- Typing status
- User connections
- User disconnections
- Message timestamps

## Notifications 🔔

If the chat tab is not active and a new message is received, the browser will display a notification.
